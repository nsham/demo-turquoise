Description

BORAL Shaftliner is developed for use in Boral Shaft Liner Cavity Wall system to provide fire protection for elevator shaft, vertical ducts and other application, when installation is posible from one side only. Boral Shaftliner is manufactured in 600mm width and recognised by its blue/ grey liner paper.
